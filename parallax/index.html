Parallax scrolling demo

Add a data-parallax-multiplier attribute to elements whose backgrounds should scroll some multiple of the actual scroll distance.

Note: I've symlinked bats/pngs/ from the the zooniverse/Visual-Assets repo to ./visual-assets.
